Welcome to the Wikimedia Foundation Governance Wiki.

This public wiki is where the Wikimedia Foundation publishes governance materials such as:

  • Board of Trustees documentation
    • Minutes
    • Resolutions
  • Legal documentation
  • Organization policies
    • Related documentation
  • Security documentation

Individuals with a registered Wikimedia account are able to edit user and talk pages as well as contribute to translations.
